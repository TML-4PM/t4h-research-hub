## DECISION LOG

### Decision 1: File Patents (vs Trade Secret)
**Date:** Feb 15, 2026  
**Decision:** FILE PATENTS (Option A + C hybrid)  
**Rationale:**
- First-mover advantage expires fast (18 months)
- Licensing revenue requires patents
- Valuation multiplier (3-5x)
- Trade secrets for implementation details

### Decision 2: Open Source MCP Bridge
**Date:** Feb 15, 2026  
**Decision:** OPEN SOURCE protocol, keep implementation proprietary  
**Rationale:**
- Ecosystem growth > licensing revenue
- Example: React (open) made Facebook worth billions
- Platform lock-in via implementation

### Decision 3: Academic Publication Timing
**Date:** Feb 15, 2026  
**Decision:** File provisionals FIRST, publish AFTER priority date  
**Rationale:**
- 12-month grace period in some jurisdictions
- Academic publication = public disclosure
- Establish priority before disclosure

### Decision 4: Biometric MVP Strategy
**Date:** Feb 15, 2026  
**Decision:** Smartphone-only Phase 1, add wearables Phase 2  
**Rationale:**
- 70% accuracy without wearables (sufficient for pilot)
- $15K vs $30K cost
- Faster time to market (6 months vs 12 months)
