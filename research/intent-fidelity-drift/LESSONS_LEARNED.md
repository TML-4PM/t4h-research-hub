# IFADS Lessons Learned

| Failure | Cause | Fix | Status |
|---|---|---|---|
| Definition of Done failure | Output treated as completion instead of outcome | Add evidence completion gates and receipts | Active |
| Memory contamination | Stale assumptions override current intent | Canonical-source-first and provenance rules | Active |
| Approval loops / reopened decisions | Solved state surfaced repeatedly | Session continuity and execution-state carry forward | Active |
| Ghost completion | Work marked complete without evidence | Mandatory receipts and evidence links | Active |
| Tool path failure | Connector unavailable or blocked | Automatic fallback routing and path detection | Active |
